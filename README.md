\# 📊 Sales Analytics Dashboard



\## 📌 Project Overview



This is an interactive Sales Analytics Dashboard built using \*\*Python\*\*, \*\*Streamlit\*\*, \*\*Pandas\*\*, and \*\*Plotly\*\*.



The dashboard helps analyze sales performance, customer behavior, product performance, and regional sales using interactive filters and visualizations.



\---



\## 🚀 Features



\- Interactive KPI Cards

\- Monthly Sales Trend

\- Sales by Region

\- Sales by Category

\- Top 10 Products

\- Top 10 Customers

\- Sales by Segment

\- Dynamic Filters

\- Download Filtered Data

\- Interactive Sales Table



\---



\## 🛠️ Technologies Used



\- Python

\- Streamlit

\- Pandas

\## ▶️ How to Run



Clone the repository



```bash

git clone https://github.com/reemahr/Sales-Analytics-Dashboard.git

```



Install dependencies



```bash

pip install -r requirements.txt

```



Run the application



```bash

streamlit run app.py

```
